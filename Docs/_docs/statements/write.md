# Write

{: .example }
> Write a black pixel:
>
> ```btml
> write black
> ```

## Write - Definition and Usage

The `write` function writes a color.

## Write - Syntax

```ebnf
write color
```

## Write - Fields

Field | Description
-- | --
color | Required. `0`/`white` or `1`/`black`.
