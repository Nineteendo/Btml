# Write

{: .example }
> Write a black pixel:
>
> ```btml
> write Black
> ```

## Write - Definition and Usage

The `write` function writes a color.

## Write - Syntax

```ebnf
write color
```

## Write - Fields

Field | Description
-- | --
color | Required. `Black` or `White`.
