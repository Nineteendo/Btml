# Move

{: .example }
> Move right until the first white pixel:
>
> ```btml
> if Black move Right repeat
> ```

## Move - Definition and Usage

The `move` function moves the program.

## Move - Syntax

```ebnf
move direction
```

## Move - Fields

Field | Description
-- | --
direction | Required. `Down`, `Left`, `Right` or `Up`.
